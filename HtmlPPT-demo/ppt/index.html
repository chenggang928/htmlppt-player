Playback Modes

Two modes, two room behaviors.

PLAY

Main screen and projector show the same current slide.

PRESENTER

Audience sees only the slide. Speaker keeps notes and control.

Use this slide to make the Play vs Presenter distinction memorable: mirror the show, or separate the room.

Presenter View

Private context stays private.

The projected window contains no notes, no thumbnails, no controls. The speaker window keeps everything needed to perform.

This is the feature that makes the tool feel like PowerPoint-style presenting rather than browser playback.

Agent Skill
